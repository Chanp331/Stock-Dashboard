# Stock Circle Dashboard

Live stock dashboard and social signal prototype packaged for GitHub Pages.

## Prototype Features

- Top watchlist dashboard
- Browser-side live quote refresh
- Buy / hold / sell signal cards
- Family and friends signal feed
- Post-a-signal form with entry, target, and reason
- Group pulse summary
- Local browser saving for watchlist and posts

## Live Page

After GitHub Pages is enabled, the dashboard is available at:

`https://chanp331.github.io/Stock-Dashboard/`

## Notes

- The dashboard is currently a front-end prototype.
- Manual watchlist additions and signal posts are saved in each browser.
- Real accounts, shared group data, comments, and notifications would require a backend database.
- This prototype is for education and discussion only, not financial advice.
